#wp-starter

##WordPress starter template, built for the modern web

wp-starter make use of the following projects:

https://github.com/
http://editorconfig.org/
http://csslint.net/
http://www.jshint.com/
http://lesscss.org/
http://bower.io/
http://gruntjs.com/

##Using Grunt

    $ grunt buildcss
    $ grunt buildjs
or
    $ grunt watch
